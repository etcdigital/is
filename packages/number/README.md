# @validation-types/number

## Install

`npm i @validation-types/number` or `yarn add @validation-types/number`

## Use

ES5

```js
const isNumber = require('@validation-types/number')

isNumber(123); // true
isNumber(123); // true
isNumber('_123'); // false
```

ES5

```js
import isNumber from "@validation-types/number"

isNumber(123); // true
isNumber(123); // true
isNumber('_123'); // false
```
