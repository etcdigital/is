# @validation-types/object

## Install

`npm i @validation-types/object` or `yarn add @validation-types/object`

## Use

ES5

```js
const isObject = require('@validation-types/object')

isObject({}); // true
isObject({ ok: true }); // true
isObject(123); // false
```

ES5

```js
import isObject from "@validation-types/object"

isObject({}); // true
isObject({ ok: true }); // true
isObject(123); // false
```
