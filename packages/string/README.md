# @validation-types/string

## Install

`npm i @validation-types/string` or `yarn add @validation-types/string`

## Use

ES5

```js
const isString = require('@validation-types/string')

isString("123"); // true
isString("abc"); // true
isString(123); // false
```

ES5

```js
import isString from "@validation-types/string"

isString("123"); // true
isString("abc"); // true
isString(123); // false
```
